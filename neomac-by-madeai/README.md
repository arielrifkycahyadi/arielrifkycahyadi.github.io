# MacBook Neo Showcase

A modern, responsive Single-Page Application (SPA) showcasing the MacBook Neo with A18 Pro chip in Indonesia.

## Features

- **Modern Design**: Glassmorphism effects, smooth animations, and responsive layout
- **Utility-First CSS**: Custom utility classes for consistent styling
- **Interactive Elements**: Power toggle for theme switching, mini-game, and smooth navigation
- **Postel Certified**: Highlighting Indonesian market certification
- **Mini-Game**: "Feed the Fat Cat" interactive game with scoring system

## Project Structure

```
macbook-neo-showcase/
```

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with custom properties, flexbox, grid, and animations
- **Vanilla JavaScript (ES6+)**: DOM manipulation and interactivity
- **SVG**: Scalable vector icons

## Key Features Implemented

### Responsive Design
- Mobile-first approach
- Flexible grid layouts
- Adaptive typography and spacing

### Modern CSS Features
- CSS Custom Properties (variables)
- Backdrop filters for glassmorphism
- CSS Grid and Flexbox
- Keyframe animations
- Responsive utilities

### Interactive Components
- SPA navigation without page reloads
- Theme switching (Light/High-Performance mode)
- Mini-game with particle effects
- Smooth scroll and hover animations

### Performance Optimizations
- Zero-runtime CSS utilities
- Efficient DOM manipulation
- Intersection Observer for animations
- Debounced scroll events
